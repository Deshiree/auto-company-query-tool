📊 关于本平台

本平台自动化采集中国全国车企的公开数据，涵盖财务数据、组织架构和招聘计划。

🔄 数据更新

  • 📅 自动更新：每天北京时间凌晨2:00
  • 🤖 采集引擎：GitHub Actions
  • 🌐 数据源：AKShare / Tushare / 车企官方招聘站

📋 数据覆盖

  • 传统自主品牌：20家
  • 新势力/新能源：10家
  • 商用车：11家
  • 合资车企：16家

⚖️ 合规声明

  • ✅ 仅采集企业公开发布的数据
  • ✅ 遵守 robots.txt 和请求频率限制
  • ❌ 不采集任何个人隐私信息

💻 技术栈

  • Backend: Python + AKShare + Tushare
  • CI/CD: GitHub Actions
  • Hosting: GitHub Pages
  • Frontend: Vanilla HTML/CSS/JS
